Welcome to MissMecha’s Documentation¶

MissMecha is a Python package for simulating and evaluating missing data mechanisms. It provides structured tools for generating missingness, visualizing patterns, and evaluating imputation methods. The package is designed for researchers, educators, and developers working with incomplete data.
